# 📘 Salesforce Day 5 Notes

# 🧪 Apex Testing

Testing helps enterprise applications:
- Prevent bugs
- Improve reliability
- Ensure correct automation
- Maintain software quality

Salesforce requires test classes before deployment.

---

# ⚡ Asynchronous Apex

Async Apex processes tasks in the background.

Examples:
- Bulk emails
- Data import
- Report generation
- External synchronization

---

# 🔄 Future Methods

Used for lightweight background processing.

---

# 🏗️ Queueable Apex

Used for:
- Complex background jobs
- Better scalability
- Job chaining

---

# 📈 Scalability Thinking

Enterprise systems must support:
- Thousands of users
- Large databases
- High traffic

---

# 🛡️ Reliability Engineering

Reliable systems:
- Recover from failures
- Prevent data inconsistency
- Maintain correct business operations

---

# ⚠️ Common Enterprise Risks

- Duplicate records
- Incorrect calculations
- Notification failures
- Database inconsistency

---

# 🧠 Key Understanding

Enterprise software is different from small programs because:
- It handles large-scale operations
- Multiple systems interact together
- Reliability becomes critical
- Failures affect business operations
