Day8 Answers

## 1️⃣ What is LWC?

LWC (Lightning Web Components) is a modern Salesforce framework used to build reusable and fast user interfaces.

---

## 2️⃣ Why Salesforce Uses LWC

Salesforce uses LWC because it:
- Improves performance
- Supports reusable components
- Uses modern web standards
- Simplifies UI development

---

## 5️⃣ Frontend vs Backend Logic

| Frontend | Backend |
|---|---|
| Button click | Fee calculation |

## 6️⃣ Reflection

Modern enterprise systems use component-based UI architecture because reusable components improve scalability, maintainability, and development speed.
