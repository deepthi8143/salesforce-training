# 📘 Salesforce Day2 Notes

# ⚡ What is LWC?

LWC (Lightning Web Components) is a modern Salesforce UI framework used to build fast and reusable user interfaces.

LWC uses:
- HTML
- JavaScript
- Metadata XML

---

# 🌐 Why Salesforce Uses LWC

Salesforce uses LWC because:
- It improves performance
- Supports reusable UI
- Follows modern web standards
- Makes development modular

---

# 🧩 What is a Component?

A component is a reusable UI block that performs a specific function.

### Examples
- Header
- Navigation Bar
- Attendance Widget
- Notification Panel

# 🧱 Component Breakdown

## Student Dashboard Components

### Header Component
Displays title and navigation.

### Student Info Component
Displays student details.

### Attendance Component
Displays attendance percentage.

### Notification Component
Shows alerts and reminders.

---

# 🔄 Why Reusable Components are Useful

Reusable components:
- Reduce duplicate code
- Improve maintainability
- Save development time
- Improve scalability

# ⚖️ Frontend vs Backend Logic

| Frontend (UI) | Backend (Apex) |
|---|---|
| Button click | Fee calculation |
| Form display | Database operations |
| Notification display | Validation logic |
| User interaction | Complex business logic |
