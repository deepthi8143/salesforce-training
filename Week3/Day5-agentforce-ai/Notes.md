# 📝 Notes – Day 17

## What is Agentforce?

Agentforce is Salesforce's AI platform that allows businesses to build intelligent AI agents capable of understanding requests, accessing data, and performing actions automatically.

---

## What is an AI Agent?

An AI Agent is a software system that can:

- Understand user requests
- Access business data
- Make decisions
- Execute actions
- Provide intelligent responses

---

## Agentforce Components

### Topics

Define what the agent can handle.

Examples:

- Student Support
- Course Registration
- Placement Guidance

---

### Actions

Tasks that an agent can perform.

Examples:

- Create records
- Update information
- Send notifications
- Trigger Flows

---

## AI + Salesforce Integration

AI agents can work with:

- Salesforce Flows
- Apex Classes
- Salesforce Data
- Platform Events

---

## Enterprise AI Governance

Organizations must control:

- Data access
- Security permissions
- AI-generated actions
- Approval workflows

---

## Benefits of AI Agents

- Faster responses
- Reduced manual effort
- Better user experience
- Increased productivity
- Intelligent automation
