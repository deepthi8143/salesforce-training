# Leave Management System

## Project Overview

The Leave Management System is a Salesforce-based application designed to automate and simplify the employee leave request process. Employees can submit leave requests, track their leave status, and view leave history, while administrators can review, approve, or reject requests efficiently.

---

## Objectives

- Reduce manual paperwork
- Simplify leave request management
- Track leave status efficiently
- Generate reports and dashboards

---

## Features

### Employee Features

- Submit Leave Requests
- View Leave Status
- View Leave History

### Administrator Features

- Review Leave Requests
- Approve Leave Requests
- Reject Leave Requests
- View Reports
- View Dashboards

---

## Salesforce Components Used

- Custom Objects
- Custom Fields
- Tabs
- Page Layouts
- Approval Processes
- Reports
- Dashboards

---

## Custom Object

### Leave Request

Fields:
- Employee Name
- Employee ID
- Leave Type
- Start Date
- End Date
- Reason
- Status

---

## Approval Workflow

1. Employee submits a leave request.
2. Leave request status is automatically set to Pending.
3. Administrator reviews the request.
4. Administrator approves or rejects the request.
5. Status is automatically updated to Approved or Rejected.

---

## Project Outcome

A fully functional Leave Management System developed using Salesforce declarative tools without Apex programming.
