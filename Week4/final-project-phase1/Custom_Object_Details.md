# Leave Request Object

Object Name: Leave Request

Fields:

1. Employee Name
   - Type: Text

2. Employee ID
   - Type: Text

3. Leave Type
   - Type: Picklist
   - Values:
     - Sick Leave
     - Casual Leave
     - Vacation Leave
     - Emergency Leave

4. Start Date
   - Type: Date

5. End Date
   - Type: Date

6. Reason
   - Type: Long Text Area

7. Status
   - Type: Picklist
   - Values:
     - Pending
     - Approved
     - Rejected
