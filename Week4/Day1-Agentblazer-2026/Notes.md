# Notes – Day 19

## What is Agentforce?

Agentforce is Salesforce's AI platform for building intelligent agents that can understand requests, access business data, and perform actions automatically.

---

## What is an AI Agent?

An AI Agent is a software system capable of:

- Understanding requests
- Retrieving information
- Making decisions
- Executing actions
- Assisting users

---

## Agentforce Components

### Topics
Define what the agent can handle.

Example:
- Student Support
- Course Registration
- Attendance Queries

---

### Actions

Actions allow agents to perform tasks.

Examples:
- Create records
- Update records
- Send notifications
- Launch Flows

---

## Benefits of Agentforce

- Faster operations
- Reduced manual work
- Better productivity
- Improved customer support
- Intelligent decision assistance

---

## Enterprise AI Challenges

- Incorrect responses
- Privacy concerns
- Security risks
- Data quality issues
- Over-automation risks

---

## Best Practices

- Validate AI outputs
- Maintain security controls
- Use human approvals when required
- Monitor AI performance regularly
