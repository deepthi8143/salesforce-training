# 📘 Salesforce Day 1 Notes

# What is Salesforce?

Salesforce is a cloud-based CRM platform that helps businesses manage customer relationships, sales, support services, and business operations efficiently.

Salesforce allows companies to:
- Store customer information
- Track leads and opportunities
- Manage support cases
- Generate reports and dashboards
- Automate workflows and business tasks

It is one of the most widely used CRM platforms in the world.

---

# What is CRM?

CRM stands for Customer Relationship Management.

CRM is used to:
- Maintain customer data
- Improve customer communication
- Track sales activities
- Increase customer satisfaction
- Improve business productivity

### Real-Time Example
If a company stores customer names, phone numbers, emails, and purchase history in one system, that system is called CRM.

# Salesforce Objects and Records

## Object

An Object is a database table used to store information in Salesforce.

### Examples
- Account
- Contact
- Lead
- Opportunity

---

## Record

A Record is a single entry inside an object.

### Example

| Name | Phone |
|---|---|
| Rahul | 9876543210 |

This single row is called a Record.

---

# Types of Objects

## Standard Objects

Objects already provided by Salesforce.

### Examples
- Account
- Contact
- Lead
- Opportunity

These are used in most business applications.

---

## Custom Objects

Objects created according to business requirements.

### Examples
- Student Object
- Employee Object
- Library Object

Custom objects help companies build their own applications.

---

# Salesforce Developer

A Salesforce Developer:
- Develops custom applications
- Writes Apex code
- Builds Lightning Components
- Creates automation
- Integrates external applications

### Skills Required
- Apex
- SOQL
- HTML/CSS
- JavaScript
- Problem-solving

---

# Difference Between Admin and Developer

| Salesforce Admin | Salesforce Developer |
|---|---|
| Configures Salesforce | Develops applications |
| Uses clicks and tools | Writes code |
| Manages users | Builds custom functionality |
| Creates reports | Creates automation |

---

# Salesforce Data Model

Salesforce stores data using:

### Example

This relationship helps organize customer information properly.
