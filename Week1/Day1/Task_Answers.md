## 1. What problem does Salesforce solve?

Salesforce helps businesses manage customer data, sales, marketing, and customer support efficiently in one platform.

It reduces manual work and improves business productivity through automation and centralized data management.

---

## 2. What is CRM?

CRM stands for Customer Relationship Management.

CRM is a system used to manage customer interactions, store customer data, track communication, and improve customer satisfaction.

---

## 3. What is an Object in Salesforce?

An Object is a database table used to store information in Salesforce.

### Example
Contact Object stores customer details like:
- Name
- Phone Number
- Email

---

## 4. Difference between Salesforce Admin and Developer

### Salesforce Admin
- Configures Salesforce
- Manages users
- Creates reports and dashboards
- Uses Salesforce tools without coding

### Salesforce Developer
- Builds applications
- Writes Apex code
- Creates Lightning Components
- Develops automation and integrations

---

## 5. Suggest one real-world application built using Salesforce.

### Hospital Management System

A Hospital Management System built using Salesforce can:
- Store patient records
- Manage doctor appointments
- Track treatments
- Handle billing and support services
