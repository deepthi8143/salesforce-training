Day 6 Answers

## 1️⃣ What is SOQL?

SOQL (Salesforce Object Query Language) is used to retrieve records from Salesforce objects.

---

## 2️⃣ What is an Apex Trigger?

An Apex Trigger is code that executes automatically when data changes occur in Salesforce.

---

## 3️⃣ Flow vs Trigger

| Flow | Trigger |
|---|---|
| No-code automation | Code-based automation |
| Easy workflows | Complex business logic |
| Visual builder | Apex programming |

---

## 4️⃣ Before vs After Trigger

| Before Trigger | After Trigger |
|---|---|
| Runs before saving | Runs after saving |
| Used for validations | Used for notifications |
| Updates values before save | Works on saved records |

---

## 5️⃣ Trigger Use Cases

### After Student Registration
Send welcome email automatically.

### After Course Full
Notify faculty members.

### After Attendance Drops
Send warning notification.

### After Fee Payment
Generate receipt automatically.

### After Student Update
Update related records.

---

## 6️⃣ Query Examples

- Find all students in Course A
- Find students with attendance below 75%
- Find all courses handled by Faculty X
- Find students with pending fees
- Find courses with available seats

## 7️⃣ Reflection

Enterprise systems need event-driven behavior because systems must react automatically to data changes and business events in real time.
