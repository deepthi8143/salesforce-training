# 📘 Salesforce Day 5 Notes

# What is Apex?

Apex is a programming language used in Salesforce to write custom business logic.

It runs on the Salesforce cloud and helps handle complex operations.

---

# Why Apex is Needed

- Flows are limited for complex logic
- Large data processing needs code
- External system integration needs Apex
- Advanced rules require programming

---

# Declarative vs Programmatic

## Declarative (No Code)
- Flow Builder
- Validation Rules
- Formula Fields

## Programmatic (Code)
- Apex Classes
- Triggers
- SOQL & DML

---

# Apex Basics

- Variables store data
- Classes define structure
- Methods perform actions
- Conditions control logic
- Loops repeat actions

---

# Database in Apex

## SOQL
Used to fetch records from Salesforce objects

## DML
Used to insert, update, delete records
