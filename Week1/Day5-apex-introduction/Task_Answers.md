#  Day 5 Answers

## 1. What is Apex?

Apex is a programming language used in Salesforce to implement complex business logic and backend operations.

---

## 2. Flow vs Apex

| Flow | Apex |
|---|---|
| No-code tool | Programming language |
| Easy automation | Complex logic |
| Limited control | Full control |

---

## 3. Configuration vs Coding

| Configuration | Coding |
|---|---|
| Click-based setup | Written code |
| Easy to use | More powerful |
| Limited logic | Advanced logic |

---

## 4. Real Examples Where Apex is Needed

- Payment gateway integration
- Complex fee calculations
- Large data processing

---

## 5. Integrated System Design

College Management System:

- CRM: Student admission process
- Objects: Student, Course, Faculty
- Relationships: Student ↔ Course
- Validation: Email required
- Flow: Auto email on registration
- Apex: Eligibility rules, complex logic
