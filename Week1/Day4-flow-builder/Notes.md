# 📘 Salesforce Day 4 Notes

## What is Flow Builder?

Flow Builder is a Salesforce tool used to automate business processes without coding.

It allows users to:
- Create workflows visually
- Automate tasks
- Reduce manual work

---

## Why Automation is Important

- Saves time
- Reduces human errors
- Improves productivity
- Ensures consistency
- Scales business processes

---

## Types of Flows

### 1. Screen Flow
- User interacts with screens
- Takes input from users
- Used for guided processes

### 2. Record-Triggered Flow
- Runs automatically
- Triggered on create/update/delete
- No user input required

---

## Flow Components

- Trigger
- Conditions
- Actions
- Decisions
- Outcomes

---

## Real-World Example

Student enrollment automation:
- Automatically assign ID
- Update records
- Send confirmation email
